# Фонд оценочных средств по дисциплине «Интеллектуальные технологии ролевого управления в многоагентных средах»

> **Статус:** шаблон репозитория. Замените все маркеры `[ЗАПОЛНИТЬ]` и удалите учебные примеры перед публикацией.

Этот репозиторий предназначен для открытого и воспроизводимого представления фонда оценочных средств (ФОС) по дисциплине «Интеллектуальные технологии ролевого управления в многоагентных средах». Здесь объединены рабочая программа дисциплины, модель измерения результатов обучения, контрольно-измерительные материалы (КИМ), критерии оценивания, методические рекомендации и используемые образовательные ресурсы. Результаты обучения связываются с компетентностно-ролевой моделью в области искусственного интеллекта (КРМ версии 3.0).

## 4. Итоговая оценка

Для допуска к экзамену студент должен быть аттестован по каждому модулю курса. Студент считается аттестованным по разделу, если освоил не менее 60% материала раздела, включая вопросы и задания, отмеченные как составляющие базовые минимум по разделу. Степень освоения раздела отражается оценкой по шкале, составляющей долю от 100-бальной шкалы:

* 1 модуль -- до 20 баллов, студент считается аттестованным, если набрано не менее 12 баллов;
* 2 модуль -- до 20 баллов, студент считается аттестованным, если набрано не менее 12 баллов;
* 3 модуль -- до 20 баллов, студент считается аттестованным, если набрано не менее 12 баллов;
* 4 модуль -- до 20 баллов, студент считается аттестованным, если набрано не менее 12 баллов.

Оценка за экзамен: до 20 баллов, положительной считается оценка не менее 12 баллов.

Итоговая оценка по курсу суммируется из оценок по разделам и за экзамен, а также отдельная оценка выставляется за курсовую работу. Для оценивания используются две шкалы: классическая "пятибальная", а также 100-бальная. Курсовая работа также оценикается по классической "пятибальной" и 100-бальной шкалаам.

Студент считается аттестованным по курсу, если он аттестован по каждому разделу и получил положительную оценка за экзамен, а суммарный балл равен не менее 60. Студент считается аттестованным по курсовой работе, если он получил за нее оценку не менее 60 баллов.

Методика оценивания курсовой работы, экзамена, а также каждого раздела приводится в соответсвующем разделе.

Таблица соответствия между "пятибальной" и 100-бальной шкалаами:

| 100-бальная оценка | оценка по "пятибальной" шкале | сокр. обозн. | код оценки |
|--------------------|-------------------------------|--------------|------------|
| 0  -- 59           | неудовлетворительно           | неуд         | 2          |
| 60 -- 69           | удовлетворительно             | уд           | 3          |
| 70 -- 89           | хорошо                        | хор          | 4          |
| 90 -- 100          | отлично                       | отл          | 5          |

## 7. Порядок заполнения

1. Заполните РПД и сформулируйте проверяемые результаты обучения.
2. Выберите релевантные профессиональные роли и компетенции КРМ.
3. Для каждого результата выберите индикатор и требуемый уровень.
4. Заполните модель измерения.
5. Создайте КИМ и рубрики оценивания по шаблонам.
6. Согласуйте систему итогового балла.
7. Добавьте ресурсы, сведения о команде и лицензию.
