# Методические указания обучающимся

## Общие рекомендации

Учебно-методический комплекс предназначен для формирования практических навыков разработки интеллектуальных агентных систем на основе больших языковых моделей (Large Language Models, LLM). Изучение материалов рекомендуется выполнять последовательно, переходя к следующему модулю только после успешного освоения предыдущего.

Перед началом работы рекомендуется:

- ознакомиться со структурой репозитория;
- установить программное обеспечение и Python-библиотеки, указанные в разделе [Python-библиотеки и программные средства](../../resources/software/python-libs/README.md);
- изучить описание соответствующего модуля и критерии оценивания лабораторной работы;
- проверить работоспособность программного окружения.

---

## Порядок изучения материалов

### [Модуль 1. Интеллектуальные агенты](../../M1-Intelligent-agent/README.md)

При изучении первого модуля рекомендуется:

1. ознакомиться с теоретическим материалом;
2. изучить архитектуру интеллектуального агента;
3. выполнить лабораторные работы в соответствии с КИМ;
4. самостоятельно изменить предложенные примеры и проанализировать результаты.

После завершения модуля обучающийся должен понимать принципы взаимодействия больших языковых моделей с пользователем и уметь создавать простейших интеллектуальных агентов.

---

### [Модуль 2. Инструменты, память и RAG](../../M2-tools-knowledge-guardrails/README.md)

Во втором модуле рекомендуется:

- изучить механизм Tool Calling;
- ознакомиться с принципами Retrieval-Augmented Generation (RAG);
- выполнить лабораторные работы по интеграции языковой модели с внешними инструментами;
- сравнить результаты работы различных вариантов реализации.

Особое внимание рекомендуется уделить анализу ограничений больших языковых моделей и способам повышения качества ответов.

---

### [Модуль 3. Мультиагентные системы](../../M3-multi-agent-systems/README.md)

При выполнении лабораторных работ третьего модуля рекомендуется:

- изучить архитектуру мультиагентных систем;
- проанализировать способы распределения ролей между агентами;
- выполнить практические задания по построению агентных графов;
- сравнить различные схемы взаимодействия интеллектуальных агентов.

При выполнении экспериментов рекомендуется фиксировать полученные результаты и анализировать влияние архитектуры системы на качество решения задач.

---

### [Модуль 4. Оценка качества и эксплуатация](../../M4-evaluation-and-operations/README.md)

В заключительном модуле рекомендуется:

- изучить методы оценки качества больших языковых моделей;
- выполнить лабораторные работы по анализу результатов;
- ознакомиться с метриками оценки качества;
- провести сравнительный анализ различных конфигураций интеллектуальных агентов.

Результаты экспериментов рекомендуется оформлять в виде таблиц и кратких аналитических выводов.

---

## Выполнение лабораторных работ

При выполнении лабораторных работ рекомендуется придерживаться следующей последовательности действий:

1. Изучить цель работы.
2. Ознакомиться с теоретическими сведениями.
3. Подготовить программное окружение.
4. Выполнить задания, приведённые в КИМ.
5. Проверить корректность работы программного кода.
6. Ответить на контрольные вопросы.
7. Сформулировать выводы по результатам выполнения работы.

---

## Самостоятельная работа

Для углубленного изучения дисциплины рекомендуется:

- изучить материалы разделов [Учебники](../../resources/textbooks/README.md) и [Научные статьи](../../resources/papers/README.md);
- выполнить дополнительные эксперименты с параметрами языковых моделей;
- самостоятельно модифицировать предложенные примеры;
- ознакомиться с официальной документацией используемых библиотек.

---

## Рекомендации по оформлению результатов

При подготовке отчётов по лабораторным работам рекомендуется включать:

- цель работы;
- описание выполненных действий;
- листинг разработанного программного кода;
- результаты выполнения программы;
- ответы на контрольные вопросы;
- выводы по результатам работы.

---

## Требования к программному окружению

Для выполнения лабораторных работ рекомендуется использовать:

- Python версии 3.12 или выше;
- актуальные версии библиотек, указанных в разделе [Python-библиотеки и программные средства](../../resources/software/python-libs/README.md);
- Jupyter Notebook или Visual Studio Code;
- Git для получения актуальной версии материалов курса.

Перед выполнением лабораторной работы рекомендуется обновить зависимости проекта и убедиться в корректной работе всех примеров.

---

### [Выполнение курсовой работы](../../Project/README.md)

Курсовая работа выполняется с целью закрепления теоретических знаний и формирования практических навыков разработки интеллектуальных систем на основе больших языковых моделей. При выборе темы рекомендуется ориентироваться на материалы модулей курса, связанные с разработкой интеллектуальных агентов, использованием внешних инструментов, технологиями Retrieval-Augmented Generation (RAG), построением мультиагентных систем или оценкой качества работы языковых моделей.

При выполнении курсовой работы обучающемуся рекомендуется использовать официальную документацию программных библиотек, учебную литературу и научные публикации, приведённые в соответствующих разделах учебно-методического комплекса. Разрабатываемое программное средство должно сопровождаться пояснительной запиской, содержащей описание постановки задачи, архитектуры решения, используемых технологий, результатов тестирования и выводов. Исходный код рекомендуется размещать в системе контроля версий Git с соблюдением требований к структуре проекта и документированию.

---

### [Подготовка к экзамену](../../Exam/README.md)

Подготовку к экзамену рекомендуется осуществлять последовательно, повторяя материалы каждого модуля курса. Особое внимание следует уделить пониманию архитектуры интеллектуальных агентных систем, принципов взаимодействия больших языковых моделей с внешними инструментами, технологии Retrieval-Augmented Generation (RAG), организации мультиагентного взаимодействия, а также методов оценки качества и эксплуатации интеллектуальных систем.

При подготовке рекомендуется повторить теоретический материал, выполнить лабораторные работы повторно, ответить на контрольные вопросы, проанализировать примеры программного кода и ознакомиться с рекомендованной литературой. Для успешной сдачи экзамена обучающийся должен уметь объяснить принципы работы рассматриваемых технологий, обосновать выбор используемых библиотек и инструментов, а также продемонстрировать практические навыки разработки и анализа интеллектуальных агентных приложений.

---

## Академическая добросовестность

Обучающийся обязан самостоятельно выполнять лабораторные работы и практические задания. Допускается использование официальной документации, учебной литературы и научных публикаций, рекомендованных в составе учебно-методического комплекса.

При использовании генеративных моделей искусственного интеллекта рекомендуется самостоятельно анализировать полученные результаты, проверять корректность сгенерированного программного кода и указывать факт использования таких инструментов в отчёте, если это предусмотрено требованиями дисциплины.
