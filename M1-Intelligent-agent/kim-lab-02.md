# КИМ-1.2. Лабораторная работа "Сранительная характеристика БЯМ"

Лабораторная работа по экспериментальному исследованию поведения БЯМ: сравнение локального инференса и обращения к внешнему API по критериям конфиденциальности данных, стоимости и времени отклика; эксперименты с токенизацией, температурой и стратегиями сэмплирования; воспроизведение и интерпретация типовых отказов модели.
