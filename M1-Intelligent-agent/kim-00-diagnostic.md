# КИМ-1.0. Входная диагностика к модулю 1

**Модуль I · проводится до начала изучения тем 1–3**

> 🏷 **Тип задания:** тестовые задания закрытого типа · задания на установление соответствия и последовательности · задания с кратким свободным ответом
> 🎯 **Когнитивный уровень (таксономия Блума):** **понимание и объяснение** (входной контроль)
> 📈 **Уровень компетенции (КРМ):** частично базовый — проверяются пререквизиты, а не материал курса

## Назначение

Диагностический КИМ. Его цель — **оценить текущие возможности студентов** перед началом дисциплины: выявить, какие пререквизиты сформированы, а какие требуют повторения.

> ⚠️ **Результаты не входят в итоговую оценку по дисциплине** и не являются основанием для недопуска. Диагностика служит двум задачам: студенту — понять, что повторить; преподавателю — скорректировать темп и глубину изложения под конкретную группу.

## Задания

### Блок A. Программирование на Python

**1.** *(выбор одного варианта)* Что вернёт выражение `{"a": 1}.get("b", 0)`?

- а) `KeyError`
- б) `None`
- в) `0`
- г) `False`

**2.** *(выбор одного варианта)* Какая конструкция гарантирует, что файл будет закрыт даже при исключении?

- а) `try / except`
- б) `with open(...) as f:`
- в) `finally: f.close()` без `try`
- г) сборщик мусора закроет автоматически

**3.** *(краткий свободный ответ)* Что делает декоратор в Python и как он влияет на исходную функцию? Ответьте одним-двумя предложениями.

### Блок B. Основы машинного обучения

**4.** *(выбор одного варианта)* Модель показывает 99 % точности на обучающей выборке и 62 % на отложенной. Как называется это явление?

- а) недообучение
- б) переобучение
- в) утечка целевой переменной
- г) дисбаланс классов

**5.** *(краткий свободный ответ)* Что такое векторное представление (эмбеддинг) текста и почему близость векторов используется для поиска похожих документов?

### Блок C. Теория вероятностей и статистика

**6.** *(выбор одного варианта)* Эксперимент повторили 5 раз и получили значения 10, 12, 11, 47, 10. Какая мера центральной тенденции устойчивее к выбросу?

- а) среднее арифметическое
- б) медиана
- в) сумма
- г) дисперсия

**7.** *(краткий свободный ответ)* Почему при сравнении двух недетерминированных систем недостаточно одного запуска каждой? Что нужно измерить дополнительно?

### Блок D. API и форматы данных

**8.** *(выбор нескольких вариантов)* Что из перечисленного является корректным JSON?

- а) `{"id": 1, "name": "тест"}`
- б) `{'id': 1}`
- в) `{"items": [1, 2, 3]}`
- г) `{"id": 1,}`

**9.** *(установление соответствия)* Сопоставьте HTTP-код и его смысл:

| Код | | Значение |
|---|---|---|
| 1) 200 | | А) ошибка на стороне сервера |
| 2) 401 | | Б) успешный запрос |
| 3) 429 | | В) не пройдена аутентификация |
| 4) 500 | | Г) превышен лимит частоты запросов |

### Блок E. Знакомство с БЯМ и промпт-инженерией

**10.** *(выбор одного варианта)* Что языковая модель непосредственно предсказывает при генерации ответа?

- а) смысл предложения целиком
- б) следующий токен на основе контекста
- в) ответ из базы знаний, где он хранится
- г) наиболее частую фразу из обучающих данных

**11.** *(установление последовательности)* Расставьте этапы обработки запроса языковой моделью в правильном порядке:

- А) генерация выходных токенов по одному
- Б) разбиение текста запроса на токены
- В) сборка токенов в итоговый текст ответа
- Г) обработка контекста моделью

**12.** *(краткий свободный ответ)* Приведите пример задачи, где применение языковой модели **неоправданно**, и объясните, чем её следует решать.

## Ключи и эталонные ответы

| 3 | Декоратор — функция, которая принимает другую функцию и возвращает новую, добавляя поведение **без изменения тела** исходной | 2 б. — упомянуты обёртка и неизменность оригинала; 1 б. — только «добавляет поведение» |
| 4 | **б)** переобучение | |
| 5 | Числовой вектор, отражающий смысл текста; семантически близкие тексты дают близкие векторы, поэтому близость (например, косинусная) работает как мера смыслового сходства | 2 б. — есть и «вектор», и «близость = сходство смысла» |
| 6 | **б)** медиана | ключевой навык для модулей 1 и 4 |
| 7 | Результат недетерминированной системы меняется от запуска к запуску; нужны **повторы** и оценка **разброса** (медиана, min–max, дисперсия), иначе разница может быть случайной | 2 б. — названы и повторы, и разброс |
| 8 | **а)** и **в)** | б) — одинарные кавычки, г) — висячая запятая |
| 9 | 1–Б, 2–В, 3–Г, 4–А | 2 б. — все верно; 1 б. — три из четырёх |
| 10 | **б)** следующий токен | центральное понятие темы 2 |
| 11 | **Б → Г → А → В** | |
| 12 | Любая операция с точным правилом и структурированным входом: проверка формата, арифметика, выборка из БД. Решается алгоритмом (регулярное выражение, SQL, формула) | 2 б. — приведён корректный пример **и** назван алгоритмический способ |
