# Модуль 1. Интеллектуальный агент и его вычислительное ядро


## Темы модуля

| № | Тема | Ключевые понятия |
|---|------|------------------|
| 1 | Введение. От промптинга к распределённым мультиагентным системам | агент = исполняемая среда (runtime) + объединяющая языковую модель + системную инструкцию + инструменты + память + механизмы защиты и планирования |
| 2 | Большая языковая модель как вычислительное ядро агента | представление БЯМ, промпты, токены; обучение; сильные и слабые стороны БЯМ; применимость |
| 3 | Базовый агент: исполняемая среда, цикл работы и ролевая системная инструкция | системная инструкция (системный промпт) как носитель роли агента: цель, ограничения, стилистические требования, политика принятия решений. Уровни сообщений диалога: system / user / assistant / tool |
