# Модуль 1. Интеллектуальный агент и его вычислительное ядро

**Лабораторная работа №2** [[ссылка]](M1-intelligent-agent/kim-lab-02.md) Лабораторная работа по экспериментальному исследованию поведения БЯМ: сравнение локального инференса и обращения к внешнему API по критериям конфиденциальности данных, стоимости и времени отклика; эксперименты с токенизацией, температурой и стратегиями сэмплирования; воспроизведение и интерпретация типовых отказов модели.

**Лабораторная работа №3** [[ссылка]](M1-intelligent-agent/kim-lab-03.md) Реализация базового агента с явным описанием роли в системной инструкции; построение графа исполнения; вывод полной трассы шагов исполнения.

**Аттестация по разделу** [[ссылка]](M1-intelligent-agent/kim-test.md) Проверочная работа на знание алгоритмов и определений.
