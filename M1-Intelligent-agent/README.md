# Модуль 1. Интеллектуальный агент и его вычислительное ядро

Список КИМ по заданиям Модуля 1:

**Лабораторная работа №1** [ссылка] Аналитический обзор: выбор предметной области; описание сценария, в котором ролевое распределение функций между агентами является обоснованным; аргументация выбора по критерию «детерминированный алгоритм / одиночный агент / мультиагентная система».

**Лабораторная работа №2** [ссылка] Лабораторная работа по экспериментальному исследованию поведения БЯМ: сравнение локального инференса и обращения к внешнему API по критериям конфиденциальности данных, стоимости и времени отклика; эксперименты с токенизацией, температурой и стратегиями сэмплирования; воспроизведение и интерпретация типовых отказов модели.

**Лабораторная работа №3** [ссылка] Реализация базового агента с явным описанием роли в системной инструкции; построение графа исполнения; вывод полной трассы шагов исполнения.
