# Постановка задачи на разработку интеллектуальной МАС: методический пример

Основные шаги:

1. Выбор предметной области: составление анатилического обзора, подготовка текстового описания предметной области и задачи

2. Декомпозиция описанной задачи, выделение подзадач/исполнителей

3. Выбор степени гранулирования, формирование перечня атомарных сущностей

4. Оценка каждой задачи по критерию применимости технологии ИИ (с обоснованием «детерминированный алгоритм / одиночный агент / мультиагентная система»)

5. Формирование требований к каждому агенту исходя из его предполагаемой роли
